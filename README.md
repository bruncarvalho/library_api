### Biblioteca

## Objetivo
Está api trata-se de uma biblioteca de livros
Linguagem JavaScript usando API REST

## Rodando o projeto
- Use o comando `npm run dev`

## Requisitos

- Node.js 22+
- Instale as dependências: `npm install`.

## Ambiente development com Docker Compose

O repositório possui um `docker-compose.yml` na raiz
Inicie os serviços: `docker compose up -d`
